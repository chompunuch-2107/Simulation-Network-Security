TCP — Transmission Control Protocol
Connection-oriented
✅ มีการยืนยันการรับข้อมูล (ACK)
✅ ส่งข้อมูลซ้ำเมื่อสูญหาย

❌ ช้ากว่า UDP
📌 ใช้กับ: HTTP, FTP, Email
UDP — User Datagram Protocol
Connectionless
❌ ไม่มีการยืนยัน
❌ ไม่ส่งซ้ำ
❌ ไม่จัดลำดับ
✅ เร็วกว่า
✅ Overhead น้อย
📌 ใช้กับ: DNS, Video Streaming, VoIP

    📋 ขั้นตอนการตั้งค่า Printer Sharing ใน Windows
    • เปิด Control Panel → Network and Sharing Center
    • คลิก Change advanced sharing settings และเปิด File and Printer Sharing
    • ไปที่ Devices and Printers → คลิกขวาเครื่องพิมพ์ → Printer Properties
    • ไปที่แท็บ Sharing → ติ๊ก Share this printer → ตั้งชื่อ Share Name
    • บนเครื่อง Client: เปิด File Explorer → พิมพ์ \\PC-Host ใน Address Bar
    • คลิกขวาเครื่องพิมพ์ที่เห็น → Connect → รอติดตั้ง Driver อัตโนมัติ
    • ทดสอบพิมพ์จากแอปพลิเคชัน → เลือกเครื่องพิมพ์ที่เพิ่งเพิ่ม
    🔍 ปัญหาที่พบบ่อยและวิธีแก้ไข
    ⚠️ ไม่เห็นเครื่องพิมพ์ที่แชร์
    แก้ไข: ตรวจสอบว่า Firewall อนุญาต File and Printer Sharing, ทั้ง 2 เครื่องต้องอยู่ใน Workgroup เดียวกัน, และเปิด Network Discovery
    ❌ Access Denied เมื่อเชื่อมต่อ
    แก้ไข: ใส่ Username/Password ของเครื่อง Host, หรือเพิ่ม Guest account, ตรวจสอบ Share Permission และ Security Permission
    ✅ วิธีป้องกันปัญหา
    กำหนดสิทธิ์เฉพาะ User ที่จำเป็น, ใช้ Static IP สำหรับ Host, เปิด Windows Update ให้ Driver ทันสมัย

    📚 สถานการณ์ตัวอย่าง IP Classes
    📋 ขั้นตอนการตั้งค่า Static IP ใน Windows
    • คลิกขวา Start → Network Connections (หรือ Settings → Network)
    • คลิกขวาที่ Ethernet / Wi-Fi → Properties
    • เลือก Internet Protocol Version 4 (TCP/IPv4) → คลิก Properties
    • เลือก Use the following IP address แล้วกรอก IP, Subnet Mask, Gateway
    • กรอก DNS Server (แนะนำ 8.8.8.8 หรือ IP ของ DNS ภายใน)
    • กด OK → ทดสอบด้วย Command Prompt: พิมพ์ ipconfig และ ping 8.8.8.8
